
WarO_Rust
=========

* a Rust submission for War-O as a code exercise

Usage:
---------

* assumes Rust is installed
* to compile: `cargo check`
    - see `cc.sh`
* to test: `cargo test`
    - see `ct.sh`
* to run: `cargo run [json config file]`
    - see `cr.sh`
